# Atividade DIO - POO Java (iPhone)

Projeto Java com modelagem orientada a interfaces para representar as funcoes do iPhone:

- Reprodutor Musical
- Aparelho Telefonico
- Navegador na Internet

## Estrutura

- `src/ReprodutorMusical.java`
- `src/AparelhoTelefonico.java`
- `src/NavegadorInternet.java`
- `src/IPhone.java`
- `src/Main.java`

## Como executar

```powershell
Push-Location "D:\abstração\Atividade\src"
javac *.java
java Main
Pop-Location
```
